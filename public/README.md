# OMI favicon package

Upload every file in this directory to the public web root of openmanuscript.org.

For Docusaurus, place the files in:

    static/

Docusaurus copies the contents of `static/` to the site root during build.

Add the contents of `head-snippet.html` to the site's `<head>` configuration.
For Docusaurus, the links can be configured in `docusaurus.config.js`.

Included:
- favicon.ico (16–256 px embedded)
- favicon PNG files
- Apple Touch Icon
- Android/PWA icons
- Windows tile icon and browserconfig.xml
- Safari pinned-tab SVG
- site.webmanifest
- ready-to-copy HTML head snippet
