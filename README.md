# Open Manuscript Studio

Alpha reference implementation for the Open Manuscript Initiative Studio.

## Alpha 0.1 scope

- React + TypeScript + Vite application shell
- Three-panel scholarly editor layout
- Native OMI manuscript object model
- Local in-browser manuscript state
- Sample manuscript creation
- Export to `.omi.json`

## Install

```bash
npm install
npm run dev
```

Open the URL printed by Vite, usually `http://localhost:5173`.

## Build

```bash
npm run build
```

The production output is generated in `dist/`.
